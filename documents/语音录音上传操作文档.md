# 语音录音上传操作文档

## 概述

本系统支持上传自定义 MP3 录音来替代 TTS 合成语音，用于声母和韵母的发音教学。

- **声母**：每个上传 1 个录音（共 23 个）
- **韵母**：每个声调各上传 1 个录音（共 34 个韵母 × 4 声调 = 136 个）
- 总计需要上传 **159 个** MP3 文件

---

## 访问地址

| 页面 | 地址 | 密码 |
|------|------|------|
| 录音上传管理 | http://localhost:3003/admin-voice-upload.html | `admin456` |
| 录音播放测试 | http://localhost:3003/voice-test.html | 无需登录 |

---

## 录音要求

| 项目 | 建议 |
|------|------|
| 格式 | MP3（.mp3） |
| 大小 | 单个文件不超过 5MB |
| 时长 | 1~3 秒为宜 |
| 采样率 | 44.1kHz 或以上 |
| 录音内容 | 清晰朗读对应的声母/韵母发音 |

---

## 操作步骤

### 1. 启动服务器

```bash
cd fun-kids
node server.js
```

### 2. 打开上传页面

浏览器访问 `http://localhost:3003/admin-voice-upload.html`，输入管理员密码 `admin456` 登录。

### 3. 上传声母录音

页面上方"声母"区域显示 23 个声母卡片：

```
b  p  m  f  d  t  n  l  g  k  h
j  q  x  zh ch sh r  z  c  s  y  w
```

操作：
1. 点击某个声母卡片上的 **「上传」** 按钮
2. 选择对应的 MP3 文件
3. 上传成功后卡片变为绿色边框，显示"已上传"
4. 点击 **「试听」** 确认录音正确

### 4. 上传韵母录音

页面下方"韵母"区域，每个韵母有 4 个声调槽位（1声 2声 3声 4声）：

操作：
1. 找到目标韵母（如 `a`）
2. 点击对应声调旁的 **「传」** 按钮
3. 选择该声调的 MP3 文件
4. 点击 **「听」** 验证

### 5. 检查上传进度

页面顶部状态栏显示：`声母 X/23 · 韵母 X/136`

### 6. 测试播放

打开 `http://localhost:3003/voice-test.html`，点击任意卡片播放录音。灰色卡片表示尚未上传。

## 文件命名规则（自动处理）

文件名由系统自动生成，存储在 `uploads/voice/` 目录下：

| 类型 | 命名格式 | 示例 |
|------|---------|------|
| 声母 | `initial-{拼音}.mp3` | `initial-b.mp3`、`initial-zh.mp3` |
| 韵母 | `final-{拼音}-{声调}.mp3` | `final-a-1.mp3`、`final-ai-3.mp3` |

---

## 韵母完整列表

| 分类 | 韵母 |
|------|------|
| 单韵母 | a o e i u ü |
| 复韵母 | ai ei ui ao ou iu ie üe er |
| 前鼻韵母 | an en in un ün |
| 后鼻韵母 | ang eng ing ong |
| 复合韵母 | ia iao ian iang iong ua uo uai uan uang |

每个韵母需录制 4 个声调，例如韵母 `a`：

| 声调 | 文件名 | 发音 |
|------|--------|------|
| 1声（阴平） | `final-a-1.mp3` | ā |
| 2声（阳平） | `final-a-2.mp3` | á |
| 3声（上声） | `final-a-3.mp3` | ǎ |
| 4声（去声） | `final-a-4.mp3` | à |

## 常见问题

**Q: 上传失败怎么办？**
A: 确认文件是 MP3 格式且不超过 5MB。检查服务器是否正常运行。

**Q: 可以替换已上传的录音吗？**
A: 可以，直接再次点击「上传」选择新文件，会自动覆盖旧文件。

**Q: 可以删除录音吗？**
A: 在上传管理页面，已上传的条目会显示「删除」按钮。

**Q: 录音存储在哪里？**
A: 服务器的 `uploads/voice/` 目录下，可直接通过文件管理器查看。

**Q: Token 过期了怎么办？**
A: 语音录音功能不依赖百度 Token，使用的是本地 MP3 文件，不会过期。
