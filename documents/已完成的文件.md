# Fun Kids 拼音学习系统

一个面向 4-6 岁儿童的互动式拼音学习系统，包含学习端与管理后台，支持 JWT 登录认证、拼音组合交互、百度 TTS 语音朗读、本地录音播放与数据管理功能。

---

## 项目结构

```
fun-kids/
├── server.js                      # Express 5 服务器（~400 行，含全部 API 路由）
├── config.js                      # 密码、JWT 密钥、百度 TTS 配置
├── package.json                   # Node.js 依赖配置
│
├── data/
│   ├── fun-kids.db                # SQLite 数据库（WAL 模式，运行后自动生成）
│   └── pinyin-seed.js             # 103 条默认拼音种子数据（10 个分类）
│
├── uploads/
│   ├── pinyin/                    # 拼音图片（上传后自动压缩 800×800 JPEG 85%）
│   └── voice/                     # 声母/韵母 MP3 录音（当前已上传 23 个）
│
├── public/
│   ├── index.html                 # 站点地图（开发导航用）
│   ├── app/
│   │   ├── index.html             # 主页 - 课程选择入口（给孩子用）
│   │   └── pinyin.html            # 拼音乐园 - 互动学习页面（给孩子用）
│   ├── admin/
│   │   ├── pinyin.html            # 拼音数据管理（给家长用）
│   │   ├── voice-upload.html      # 录音上传管理（给家长用）
│   │   ├── voice-test.html        # 录音播放测试
│   │   └── tts-test.html          # 百度 TTS 语音测试
│   ├── libs/
│   │   ├── fontawesome/           # Font Awesome 图标库（本地部署）
│   │   └── fonts/                 # Noto Sans SC 本地字体
│   └── tools/
│       └── oauth2.0token.py       # 百度 TTS Token 刷新脚本（30 天有效期）
│
└── documents/
    ├── PLAN.md                    # 项目规划文档
    ├── 已完成的文件.md              # 本文档
    ├── 语音录音上传操作文档.md       # 语音录音上传操作指南
    └── web-design/                # 设计稿与部署脚本
```

## 启动方式

```bash
cd fun-kids
npm install      # 安装依赖（已完成可跳过）
node server.js   # 启动服务器
```

杀停服务：
```bash
lsof -i :3003 2>/dev/null
kill <PID>
```

## 已完成功能

### 后端 (server.js)

- [x] Express 5 服务器，端口 3003
- [x] SQLite 数据库（WAL 模式），自动建表、自动导入种子数据
- [x] JWT 认证中间件（7 天有效期，区分 user/admin 角色）
- [x] 拼音 CRUD API（GET/POST/PUT/DELETE）
- [x] 拼音查询支持多条件筛选（category, hasImage, search, initial, final, tone）
- [x] 图片上传 API（multer + sharp 自动压缩 800×800 JPEG 85%）
- [x] 数据导入/导出/重置 API
- [x] 语音录音 API（上传/列表/删除）
- [x] 百度 TTS 配置 API
- [x] 静态文件服务（public/ + uploads/）

### 前端 - 主页 (public/app/index.html)

- [x] 密码保护登录界面
- [x] 学科卡片选择（拼音可进入，其余"即将开放"）
- [x] 浮动云朵/星星装饰动画
- [x] 彩虹渐变标题
- [x] JWT token 存储在 localStorage

### 前端 - 拼音乐园 (public/app/pinyin.html)

- [x] 声母选择面板（23 个 + 零声母 ∅）

- [x] 组合查询 + 结果展示（拼音分解、汉字、释义、图片/Emoji）

- [x] 本地 MP3 录音播放（声母 + 韵母发音）
- [x] 朗读模式开关（选择时自动播放发音）
- [x] 撒花粒子动画（成功组合时）
